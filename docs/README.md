# 项目文档

## 项目概述

量化择时指标应用 - 基于多维度择时框架的专业投资分析工具

## 目录结构说明

```
ValueInvestingLearn/
├── app/                    # 后端应用代码
│   ├── main.py            # 主应用文件
│   ├── routes/            # API路由
│   ├── services/          # 业务逻辑服务
│   ├── models/            # 数据模型
│   └── utils/             # 工具函数
├── frontend/              # 前端应用代码
│   ├── src/
│   │   ├── components/    # React组件
│   │   ├── services/      # API调用服务
│   │   └── App.js         # 主应用组件
│   └── package.json       # 前端依赖配置
├── tests/                 # 测试代码
├── data/                  # 数据存储
├── config/                # 配置文件
├── docs/                  # 项目文档
└── requirements.txt       # Python依赖
```
